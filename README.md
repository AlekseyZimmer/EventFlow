# EventFlow (ТаймМастер)

EventFlow — это приложение для управления ходом мероприятий в реальном времени. Оно позволяет организаторам оперативно реагировать на изменения, отслеживать выполнение этапов и информировать всех участников.

## Основные возможности
- **Создание мероприятий:** Планирование событий с разбивкой на этапы, указанием времени и ответственных лиц.
- **Умный ввод локации:** Автозаполнение адресов с использованием Yandex Карт (с учетом геолокации пользователя).
- **Управление в реальном времени:** Панель организатора для запуска и завершения этапов "на лету".
- **Синхронизация участников:** Мгновенное обновление расписания у всех зрителей и участников через WebSockets.
- **Отслеживание задержек:** Автоматический расчет отставания от графика.

## Технологический стек
- **Frontend:** React 19, Vite, Tailwind CSS, React Router, Lucide React.
- **Backend:** Node.js, Express.
- **Real-time:** Socket.IO.
- **API:** Yandex Geocoder API (для поиска локаций).

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Настройте переменные окружения:
   Скопируйте `.env.example` в `.env` и укажите ваш ключ API от Яндекс Карт (HTTP Геокодер):
   ```bash
   VITE_YANDEX_API_KEY="ваш_ключ_яндекс_геокодера"
   ```
3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
4. Откройте приложение в браузере по адресу `http://localhost:3000`.
